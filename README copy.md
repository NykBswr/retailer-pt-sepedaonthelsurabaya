# Manufacturing Dashboard

## Description

This Manufacturing Dashboard is a project built using **Laravel** and **TailwindCSS** as part of my corporate application integration course assignment. The dashboard integrates 5 data sources using **Pentaho** and visualizes the results using **ApexCharts**. The project utilizes **SQLite** as the database for data management.

## Features

- Integration of 5 data sources using **Python**.
- Data visualization on the dashboard using **ApexCharts**.
- Responsive design with **TailwindCSS** for a modern user interface.
- Data management integrated with **SQLite**.
- Fully responsive and accessible on different devices.

## Technologies Used

- **Laravel**: PHP framework for web application development.
- **TailwindCSS**: CSS framework for responsive and flexible UI/UX design.
- **SQLite**: Lightweight database for managing and storing data.
- **Pentaho**: ETL (Extract, Transform, Load) tool for integrating data from multiple sources.
- **ApexCharts**: Library used for creating interactive and dynamic data charts.
